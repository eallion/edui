# 8 平均数与条形统计图

## 单元目标

1.体会平均数的作用，能计算平均数，能用自己的语言解释其实际意义。

2.认识复式条形统计图，了解复式条形统计图的特点，能根据收集的数据在提供的样图中完成相应的复式条形统计图。

3.会看复式条形统计图，能根据图中的信息提出简单的问题，进行一些分析和判断。

4.培养学生的数据分析观念、推理能力和应用意识。
