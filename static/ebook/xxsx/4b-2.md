# 2 观察物体（二）

## 单元目标

1.让学生经历观察的过程，认识到从不同的位置观察物体，所看到的形状是不同的。

2.通过观察实物，能正确辨认从正面、侧面、上面观察到的两个物体或一组立体图形的位置关系和形状。

3.通过拼搭活动，培养学生的空间想象和推理能力。
