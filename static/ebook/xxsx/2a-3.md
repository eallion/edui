# 3 角的初步认识

## 单元目标

1.结合生活情境及操作活动，使学生初步认识并会判断角，知道角的各部分的名称，初步学会用尺画角。

2.结合生活情境及操作活动，使学生初步认识直角、锐角和钝角，会用三角尺判断直角、锐角和钝角。

3.让学生运用角的知识解决简单的问题，继续培养学生解决问题的能力。

4.培养学生初步的观察能力、动手操作能力和语言表达能力，并学会从数学的角度去观察周围的世界。
