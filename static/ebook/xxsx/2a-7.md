# 7 认识时间

## 单元目标

1.在具体的生活情境中，借助钟面认识时间单位“分”，知道1时=60分。

2.结合直观演示和操作，使学生知道在钟面上分针走1“小格”是1分钟，初步认识几时几分(5分5分地数)，会读写几时几分和几时半。

3.使学生会运用时间的有关知识解决一些简单的实际问题。

4.使学生进一步学习观察、比较的方法，并形成初步的推理能力。

5.初步培养学生珍惜时间、合理安排时间的习惯。
