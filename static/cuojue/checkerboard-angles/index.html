如何/什么?

这种方格布局会欺骗你的大脑，让你误以为没有角度的地方也有角度。单击任意位置即可停止动画。

内容:这种错觉应该相当清晰，有一个棋盘图案，每个角内有一小组对角棋子，具有交替图案。然而，这种模式，切换黑白方块开始扭曲你的思维，突然间，看起来线条，特别是边缘周围的线条是有角度的，而不是完全垂直的。

我第一次遇到这种特殊效果是在一个名为 Revolution Analytics 的博客中，该博客的所有者正在使用编程语言 R 创建效果。我自己的版本是相同效果的 JavaScript 扩展，但有一些改动（例如动画，以及随时停止它的能力。）

操作:盯着这个坏男孩，试着假装你正在沿着完全平行的线直线行驶。您还可以单击任意位置来停止动画并在不移动的情况下查看效果。动画看了一会儿，停止的效果感觉格外奇怪。

解释:这个网站上有一些视错觉使用了类似的技巧，一个是螺旋，另一个是直线正方形，这两种错觉，至少根据我的研究用途偏移或有角度的正方形的组合。当你的大脑尝试处理这些信息时，它开始对重复模式或基于自然界中经常出现的模式做出假设，尽管在这种情况下，它是不正确的。
